Анимация

AnimationAction - планирует (составляет расписание, по которому происходит) выполнение анимаций, которые хранятся в AnimationClip'ах.
AnimationClip - многократно используемый набор треков ключевых кадров, представляющих анимацию.
AnimationMixer - проигрыватель для анимации конкретного объекта на сцене.
AnimationObjectGroup - группа объектов, для которой выполняется совместная анимация.
AnimationUtils - различные функции для помощи при анимации.
KeyframeTrack - последовательность ключевых кадров, согласованная по времени показа.
PropertyBinding - ссылка на реальное свойство в графе сцены.
PropertyMixer - буферное свойство графа сцены, которое позволяет весовое накопление (аккумуляцию).
