PropertyBinding

Здесь содержится ссылка на реальное свойство в графе сцены; используется внутри three.js.

Конструктор

PropertyBinding( rootNode, path, parsedPath )
rootNode -- .
path -- .
parsedPath -- (дополнительный, необязательный аргумент).

Свойства


.path


.parsedPath


.node


.rootNode


.BindingType


.Versioning


.GetterByBindingType


.SetterByBindingTypeAndVersioning

Методы


.getValue( targetArray, offset )


.setValue( sourceArray, offset )


.bind( )
Создает пару getter / setter (получатель / установщик) для свойства в графе сцены. Используется внутри three.js методами getValue и setValue.

.unbind( )
Отвязывает пару getter / setter (получатель / установщик) для свойства в графе сцены.

.Composite( targetGroup, path, optionalParsedPath )
Метод создает новый Composite PropertyBinding.

.create( root, path, parsedPath )
Метод создает новый Composite PropertyBinding (если root это AnimationObjectGroup) или PropertyBinding.

.parseTrackName( trackName )
Сопоставляет строки в следующих формах:
-- nodeName.property
-- nodeName.property[accessor]
-- nodeName.material.property[accessor]
-- uuid.property[accessor]
-- uuid.objectName[objectIndex].propertyName[propertyIndex]
-- parentName/nodeName.property
-- parentName/parentName/nodeName.property[index]
-- .bone[Armature.DEF_cog].position
-- scene:helium_balloon_model:helium_balloon_model.position

.findNode( root, nodeName )
Метод находит узел (ноду) по названию в "дереве" узлов или "скелете".
