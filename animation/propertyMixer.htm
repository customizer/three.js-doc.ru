PropertyMixer

Буферное свойство графа сцены, которое позволяет весовое накопление (аккумуляцию); используется внутри three.js.

Конструктор

PropertyMixer( binding, typeName, valueSize )
binding --
typeName --
valueSize --

Свойства


.binding


.buffer
Буфер размером valueSize * 4.
Свойство имеет такую схему: [ incoming | accu0 | accu1 | orig ]
Интерполяторы могут использовать свойство buffer в качестве своего result и затем данные передавать в incoming.
accu0 и accu1 используются с чередованием кадров для накопления результата и сравниваются для определения изменений.
orig сохраняет исходное состояние свойства.

.cumulativeWeight
Значение по умолчанию равно 0.

.valueSize


.referenceCount
Значение по умолчанию равно 0.

.useCount
Значение по умолчанию равно 0.

Методы


.accumulate( accuIndex, weight )
Аккумулирует (накапливает) данные в buffer[accuIndex] области incoming в accu[i].
Если вес (параметр weight) равен 0, то не делается ничего.

.apply( accuIndex )
Метод применяет состояние accu[i] buffer'а к привязке, когда данные в аккумуляторах различаются. 

.saveOriginalState( )
Запоминает состояние связанного свойства и копирует его в оба накопителя (accu).

.restoreOriginalState(  )
Метод применяет к привязке состояние, ранее взятое с помощью метода saveOriginalState.
