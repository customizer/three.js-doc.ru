StringKeyframeTrack

Трек строковых значений ключевых кадров.

Конструктор

StringKeyframeTrack( name, times, values )
name - (обязателен) идентификатор KeyframeTrack.
times - (обязателен) массив времен ключевого кадра.
values - значения для ключевых кадров в указанное время.
interpolation - тип используемой интерполяции. Возможные значения смотрите в статье "Анимационные константы". Значением по умолчанию является InterpolateDiscrete.

Свойства

Для ознакомления с унаследованными свойствами смотрите описание KeyframeTrack. 

.DefaultInterpolation
Тип используемой интерполяции по умолчанию, InterpolateDiscrete.

.ValueBufferType
Обычный массив (в этом случае это не Float32Array, в отличие от ValueBufferType из KeyframeTrack).

.ValueTypeName
Строка 'string' - строка.
