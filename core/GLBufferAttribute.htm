GLBufferAttribute

Данный класс буферных атрибутов сам не создает объекта буфера вершин (VBO). Вместо этого он использует любой VBO, переданный в конструктор, и который впоследствии может быть изменен с помощью свойства buffer.

Вместе с VBO необходимо передавать дополнительные параметры: контекст GL (свойство buffer), тип данных GL (type), количество компонентов на вершину (itemSize), количество байтов на компонент (elementSize) и число вершин (count).

Наиболее часто данный класс используется когда мешает какой-то расчет GPGPU или даже производит рассматриваемые VBO.

Конструктор


GLBufferAttribute( buffer, type, itemSize, elementSize, count )
buffer — значением должен быть WebGLBuffer.
type — один из типов данных WebGL.
itemSize — число значений массива, связанных с конкретной вершиной. Например, если этот атрибут хранит трехкомпонентный вектор (такой как position - положение, normal - нормаль или color - цвет), тогда itemSize должен быть равен 3.
elementSize — размер (в байтах), соответствующий заданному значению параметра type. Значением может быть 1, 2 или 4.
  ●   gl.FLOAT - 4
  ●   gl.UNSIGNED_SHORT - 2
  ●   gl.SHORT - 2
  ●   gl.UNSIGNED_INT - 4
  ●   gl.INT - 4
  ●   gl.BYTE - 1
  ●   gl.UNSIGNED_BYTE - 1
count — ожидаемое количество вершин VBO.

Свойства


.buffer
Текущий экземпляр WebGLBuffer.

.count
Ожидаемое количество вершин VBO

.itemSize
Свойство определяет количество значений, приходящихся на каждый пункт (вершину).

.elementSize
Свойство хранит соответствующий размер в байтах для значения текущего свойства type.
Список известных размеров type смотрите выше (в конструкторе).

.type
Тип данных WebGL, описывающий основное содержимое VBO.
Установите это свойство вместе с elementSize. Рекомендуется использовать метод setType.

.isGLBufferAttribute
Свойство предназначено только для чтения. Значение всегда равно true.

Методы


.setBuffer( buffer )
Метод устанавливает свойство buffer.

.setType( type, elementSize )
Устанавливает значения свойств type и elementSize.

.setItemSize( itemSize )
Устанавливает значение свойства itemSize.

.setCount( count )
Устанавливает значение свойства count.

.version
Номер версии, увеличивающийся всякий раз, когда needsUpdate устанавливается как true.

.needsUpdate
Значением по умолчанию является false. Установка значения как true увеличивает номер версии на единицу.
