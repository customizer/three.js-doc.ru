Список устаревших элементов API

Поскольку интерфейс three.js довольно быстро развивается, можно столкнуться с примерами, предлагающими использование элементов API, которые больше не являются частью основных элементов.

Ниже приведен список таких элементов, а также информация относительно их замены.

Звуковое сопровождение

Audio

Audio.load устарел, взамен используйте AudioLoader.

AudioAnalyser

AudioAnalyser.getData() был переименован в AudioAnalyser.getFrequencyData().

BinaryTextureLoader

BinaryTextureLoader был переименован в DataTextureLoader.

BufferAttribute.copyIndicesArray() был удален.

Float64Attribute был удален. Вместо него используйте Float64BufferAttribute.

Камеры

PerspectiveCamera

PerspectiveCamera.setLens() устарел, для фотографических настроек используйте PerspectiveCamera.setFocalLength() и PerspectiveCamera.filmGauge().

Константы

LinePieces

Для линий больше не поддерживается режим LinePieces. Взамен создавайте LineSegments.

Основные элементы three.js

EventDispatcher.apply был удален. Вместо него, для подмешивания используйте либо наследование, либо метод Object.assign прототипа.

Uniform.dynamic был удален. Вместо него используйте Object3D.onBeforeRender().
Uniform.onUpdate был удален. Вместо него используйте Object3D.onBeforeRender().
