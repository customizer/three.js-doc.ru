DeviceOrientationControls

Может использоваться для ориентации объекта, основываясь на местоположение двигающегося устройства.

Пример

misc / controls / deviceorientation

Конструктор

DeviceOrientationControls( object )
object - объект для управления.
Создает новый экземпляр DeviceOrientationControls.

Свойства


.alphaOffset
Смещение угла альфа в радианах. Значение по умолчанию равно 0.

.deviceOrientation
Текущий объект события deviceorientation.

.enabled
Свойство определяет, включен или нет элемент управления (контрол).

.object
Объект для управления.

.screenOrientation
Ориентация окна просмотра в градусах (с шагом 90 градусов) относительно естественной ориентации устройства. Значение по умолчанию равно 0.

Методы


.connect()
Добавляет слушателей событий элемента управления (контрола) и включает их.

.disconnect()
Удаляет слушателей событий из элемента управления (контрола) и отключает их.

.dispose()
Данный метод следует вызывать когда элемент управления (контрол) больше не требуется.

.update()
Обновляет элемент управления (контрол). Обычно вызывается в анимационном цикле.
