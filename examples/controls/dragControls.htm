      DragControls

Данный класс можно использовать для предоставления операций перетаскивания объектов.

Пример

var controls = new THREE.DragControls( objects, camera, renderer.domElement );
// add event listener to highlight dragged objects
// добавляем слушатель событий для подсветки перетаскиваемых объектов
controls.addEventListener( 'dragstart', function ( event ) {
  event.object.material.emissive.set( 0xaaaaaa );
} );

controls.addEventListener( 'dragend', function ( event ) {
  event.object.material.emissive.set( 0x000000 );
} );

Конструктор

DragControls( objects, camera, domElement )
objects
- массив перетаскиваемых 3D-объектов.
camera - камера представленной сцены.
domElement - HTML-элемент, используемый для слушателей (listener) событий.
Создает новый экземпляр DragControls.

События


dragstart
Событие срабатывает когда пользователь начинает перетаскивать 3D-объект.

drag
Происходит когда пользователь перетаскивает 3D-объект.

dragend
Происходит когда пользователь завершает перетаскивание 3D-объекта.

hoveron
Событие происходит когда указатель наведен на 3D-объект или на один из его дочерних объектов.

hoveroff
Событие происходит когда снято наведение указателя на 3D-объект.

Свойства


.enabled
Свойство определяет, включен или нет элемент управления (контрол).

.transformGroup
Данная опция работает только если массив объектов содержит отдельный перетаскиваемый групповой объект.
Если установлено значение true, DragControls изменяет положение не отдельных объектов, а группы объектов целиком. Значением по умолчанию является false.

Методы

Общие методы смотрите в описании базового класса EventDispatcher. 

.activate()
Добавляет слушатель событий элемента управления (контрола).

.deactivate()
Удаляет слушатель событий элемента управления (контрола).

.dispose()
Данный метод следует вызывать когда элемент управления (контрол) больше не требуется.
