FirstPersonControls

Данный класс является альтернативной реализацией FlyControls.

Пример

webgl / geometry / terrain

Конструктор

FirstPersonControls( object, domElement )
object - камера, которой будем управлять.
domElement - HTML-элемент, используемый для слушателей (listener) событий.
Создает новый экземпляр FirstPersonControls.

Свойства


.activeLook
Свойство определяет, есть или нет возможности осмотреться вокруг. Значением по умолчанию является true.

.autoForward
Свойство определяет, будет ли камера автоматически двигаться вперед или нет. Значением по умолчанию является false.

.constrainVertical
Определяет будет или нет круговой осмотр ограничен по вертикали посредством свойств verticalMin и verticalMax. Значением по умолчанию является false.

.domElement
Элемент DOM-структуры HTML-документа, используемый для прослушивания событий мышки/тачпада. Это свойство должно быть передано в конструкторе; его изменение здесь не приведет к созданию новых слушателей событий.

.enabled
Свойство определяет, включен или нет элемент управления (контрол). Значением по умолчанию является true.

.heightCoef
Определяет, насколько быстро движется камера, когда её y-координата находится около heightMax. Значение по умолчанию равно 1.

.heightMax
Верхний предел высоты камеры, используемый для регулировки скорости движения. Значение по умолчанию равно 1.

.heightMin
Нижний предел высоты камеры, используемый для регулировки скорости движения. Значение по умолчанию равно 0.

.heightSpeed
Влияет ли высота камеры на скорость движения вперед. Значением по умолчанию является false. Для настройки используйте свойства heightCoef, heightMin и heightMax.

.lookVertical
Свойство определяет, имеется ли возможность посмотреть вкруговую по вертикали или нет. Значением по умолчанию является true.

.lookSpeed
Скорость просмотра вокруг. Значение по умолчанию равно 0.005.

.mouseDragOn
Определяет, могут или нет нажиматься кнопки мышки. Свойство только для считывания.

.movementSpeed
Скорость передвижения. Значение по умолчанию равно 1.

.object
Камера, которой будем управлять.

.verticalMax
Насколько высоко можно смотреть по вертикали, верхний предел. Значение измеряется в радианах и должно быть в диапазоне от 0 до Math.PI. Значение по умолчанию равно Math.PI.

.verticalMin
Насколько низко можно смотреть по вертикали, нижний предел. Значение измеряется в радианах и должно быть в диапазоне от 0 до Math.PI. Значение по умолчанию равно 0.

Методы


.dispose()
Данный метод следует вызывать когда элемент управления (контрол) больше не требуется.

.handleResize()
Данный метод должен быть вызван при изменении размера окна приложения.

.lookAt( vector )
.lookAt( x, y, z )
vector - вектор, представляющий целевую позицию.
Дополнительно, можно представить положение в "мировом" пространстве компонентами вектора, координатами x, y и z.
Обеспечивает ориентацию камеры элементом управления (контролом) в направлении к заданному целевому положению.

.update( delta )
delta - значение дельты (промежутка) времени.
Обновляет элемент управления (контрол). Обычно вызывается в анимационном цикле.
