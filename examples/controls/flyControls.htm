FlyControls

FlyControls обеспечивает навигацию аналогичную режимам полета в DCC-инструментах, таких как Blender. Можно произвольно изменять положение камеры в трехмерном пространстве без каких-либо ограничений (например, сфокусироваться на конкретной цели).

Пример

misc / controls / fly

Конструктор

FlyControls( object, domElement )
object - камера, которой будем управлять.
domElement - HTML-элемент, используемый для слушателей (listener) событий.
Создает новый экземпляр FlyControls.

Свойства


.autoForward
Если установить как true, камера будет автоматически двигаться вперед (без остановки) с самого начала перемещения. Значением по умолчанию является false.

.domElement
Элемент DOM-структуры HTML-документа, используемый для прослушивания событий мышки/тачпада. Это свойство должно быть передано в конструкторе; его изменение здесь не приведет к созданию новых слушателей событий.

.dragToLook
Если установить как true, то смотреть вокруг можно только выполняя операцию перетаскивания. Значением по умолчанию является false.

.movementSpeed
Скорость движения. Значение по умолчанию равно 1.

.object
Камера, которой будем управлять.

.rollSpeed
Скорость вращения. Значение по умолчанию равно 0.005.

Методы


.dispose()
Данный метод следует вызывать когда элемент управления (контрол) больше не требуется.

.update( delta )
delta - значение дельты (промежутка) времени.
Обновляет элемент управления (контрол). Обычно вызывается в анимационном цикле.
