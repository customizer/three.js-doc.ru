      TrackballControls

TrackballControls похож на OrbitControls. Однако, он не поддерживает свойство камеры camera.up. Это означает, что когда камера находится над "северным" и "южным" полюсами, её нельзя повернуть так, чтобы правая сторона оказалась вверху.

camera.up

Пример

misc / controls / trackball

Конструктор

TrackballControls( camera, domElement )
camera - камера представленной сцены.
domElement - HTML-элемент, используемый для слушателей (listener) событий.
Создает новый экземпляр TrackballControls.

События


change
Событие происходит при изменении положения камеры элементом управления (контролом).

start
Срабатывает при начале взаимодействия (например, прикосновению к тачпаду (экрану, в случае сенсорного экрана) или движению мышки).

end
Срабатывает по завершению взаимодействия.

Свойства


.domElement
Элемент DOM-структуры HTML-документа, используемый для прослушивания событий мышки/тачпада. Это свойство должно быть передано в конструкторе; его изменение здесь не приведет к созданию новых слушателей событий.

.dynamicDampingFactor
Свойство определяет интенсивность затухания (damping). Учитывается только если свойство staticMoving установлено как false. Значение по умолчанию равно 0.2.

.enabled
Свойство определяет, будет или нет включен элемент управления (контрол).

.keys
В данном массиве хранятся коды кнопок клавиатуры для управления взаимодействием.
Если нажата клавиша, указанная первой, все действия с мышкой (нажатия на левую, среднюю и правую кнопки с перемещением самой мышки) приведут к движению по окружности (по орбите).
Если нажата клавиша, указанная второй, все действия с мышкой (нажатия на левую, среднюю и правую кнопки с перемещением самой мышки) приведут к изменению масштаба.
Если нажата клавиша, указанная третьей, все действия с мышкой (нажатия на левую, среднюю и правую кнопки с перемещением самой мышки) приведут к панорамированию, т.е. к перемещению в сторону (влево, вправо, вверх и вниз).
Значением по умолчанию являются 65, 83 и 68, которые соответственно представляют клавиши A, S и D.
Вот полный список кодов нажатий клавиш клавиатуры.

.maxDistance
Насколько сильно можно уменьшить масштаб. Значение по умолчанию равно Infinity.

.minDistance
Насколько сильно можно увеличить масштаб. Значение по умолчанию равно 0.

.mouseButtons
Данный объект содержит ссылки на действия мышки, используемые элементами управления (контролами).
.LEFT присваивается значение THREE.MOUSE.ROTATE
.MIDDLE присваивается значение THREE.MOUSE.ZOOM
.RIGHT присваивается значение THREE.MOUSE.PAN

.noPan
Свойство определяет, будет или нет отключено панорамирование. Значением по умолчанию является false.

.noRotate
Будет или нет отключено вращение. Значением по умолчанию является false.

.noZoom
Свойство определяет будет или нет отключено масштабирование. Значением по умолчанию является false.

.object
Камера, которой будем управлять.

.panSpeed
Скорость панорамирования. Значение по умолчанию равно 0.3.

.rotateSpeed
Скорость вращения. Значение по умолчанию равно 1.0.

.screen
Представляет свойства экрана. Устанавливается автоматически при вызове handleResize().
left - представляет смещение в пикселах от левой границы экрана.
top - представляет смещение в пикселах от верхней границы экрана.
width - представляет ширину экрана в пикселах.
height - представляет высоту экрана в пикселах.

.staticMoving
Определяет будет или нет отключено затухание (damping). Значением по умолчанию является false.

.zoomSpeed
Скорость масштабирования. Значение по умолчанию равно 1.2.

Методы


.checkDistances()
Обеспечивает, что элементы управления остаются в диапазоне расстояний [minDistance, maxDistance]. Вызывается методом update().

.dispose()
Данный метод следует вызывать когда элемент управления (контрол) больше не требуется.

.handleResize()
Данный метод должен быть вызван при изменении размера окна приложения.

.panCamera()
При необходимости выполняет панорамирование. Вызывается методом update().

.reset()
Сбрасывает настройки элемента управления (контрола) к первоначальному состоянию.

.rotateCamera()
При необходимости выполняет поворот камеры. Вызывается методом update().

.update()
Обновляет элемент управления (контрол). Обычно вызывается в анимационном цикле.

.zoomCamera()
При необходимости выполняет масштабирование. Вызывается методом update().
