      TransformControls

Данный класс можно использовать для преобразования объектов в 3D пространстве посредством адаптации сходных интерактивных моделей из DCC инструментов вроде Blender'а. В отличие от других элементов управления, он не предназначен для изменения камеры сцены.
Для работы TransformControls предполагается, что связанный с ним трехмерный объект является частью графа сцены.

Пример

misc / controls / transform

Конструктор

TransformControls( camera, domElement )
camera - камера представленной сцены.
domElement - HTML-элемент, используемый для слушателей (listener) событий (дополнительный, необязательный параметр). По умолчанию им является документ целиком, тем не менее, если требуется чтобы элементы управления (контролы) работали только с определенным документом (например с canvas), его можно здесь указать.
Создает новый экземпляр TransformControls.

События


change
Событие вызывается при выполнении любого типа изменения объекта или свойства. Изменения свойств являются отдельными событиями, на которые также можно добавить слушатели событий. Типом события является "propertyname-changed".

mouseDown
Срабатывает когда указатель (мышка/тачпад) становится активным, например в случае мышки - курсор находится над объектом и нажата кнопка мышки.

mouseUp
Срабатывает когда указатель (мышка/тачпад) перестает быть активным, например в случае мышки - курсор находится над объектом и отпущена кнопка мышки.

objectChange
Событие срабатывает при изменении управляемого 3D-объекта.

Свойства

Общие свойства смотрите в описании базового класса Object3D. 

.axis
Ось координат текущей трансформации.

.camera
Камера представленной сцены.

.domElement
Элемент DOM-структуры HTML-документа, используемый для прослушивания событий мышки/тачпада. Это свойство должно быть передано в конструкторе; его изменение здесь не приведет к созданию новых слушателей событий.

.dragging
Определяет будет или нет в настоящий момент выполняться перетаскивание. Свойство предназначено только для считывания.

.enabled
Свойство определяет, включен или нет элемент управления (контрол).

.mode
Текущий режим трансформации. Возможными значениями являются "translate", "rotate" и "scale". Значение по умолчанию "translate".

.object
Управляемый контролом 3D-объект.

.rotationSnap
По умолчанию, 3D объекты постоянно вращаются. Если установить данное свойство равным какому-либо числовому значению (в радианах), можно определить за сколько шагов 3D объект будет повернут. you can define in which steps the 3D object should be rotated. Значением по умолчанию является null.

.showX
Определяет, будет или нет видна вспомогательная ось X. Значением по умолчанию является true.

.showY
Определяет, будет или нет видна вспомогательная ось Y. Значением по умолчанию является true.

.showZ
Определяет, будет или нет видна вспомогательная ось Z. Значением по умолчанию является true.

.size
Размер вспомогательного UI интерфейса пользователя (оси/плоскости). Значение по умолчанию равно 1.

.space
Свойство определяет, в каком пространстве координат следует выполнять изменения. Возможными значениями являются "world" и "local". Значение по умолчанию "world".

.translationSnap
По умолчанию 3D объекты постоянно перемещаются. Если установить в данном свойстве числовое значение (в "мировых" единицах), можно определить за сколько шагов 3D объект будет перенесен. By default, 3D objects are continously translated. If you set this property to a numeric value (world units), you can define in which steps the 3D object should be translated. Значением по умолчанию является null.

Методы

Общие методы смотрите в описании базового класса Object3D. 

.attach( object )
object - 3D объект, который должен быть изменен.
Задает 3D объект для изменения и обеспечивает видимость управляющего UI (интерфейс пользователя).

.detach()
Метод удаляет текущий 3D объект из элемента управления (контрола) и делает невидимым управляющий UI (интерфейс пользователя).

.dispose()
Данный метод следует вызывать когда элемент управления (контрол) больше не требуется.

.getMode()
Возвращает установленный режим изменения.

.setMode( mode )
mode - режим изменения.
Метод устанавливает режим изменения.

.setRotationSnap( rotationSnap )
rotationSnap - The rotation snap.
Sets the rotation snap.

.setSize( size )
size - размер вспомогательного UI (интерфейса пользователя).
Метод устанавливает размер вспомогательного UI (интерфейса пользователя).

.setSpace( space )
space - пространство координат, в которых применяется трансформация (изменение).
Метод устанавливает пространство координат, в которых применяется трансформация (изменение).

.setTranslationSnap( translationSnap )
translationSnap - The translation snap.
Sets the translation snap.
