Примеры

Здесь даются примеры реализации различных подсобных объектов и функций.

MMDAnimationHelper - вспомогательный элемент для анимации с ресурсами в формате MMD
MMDPhysics - обработка физических свойств для ресурсов MMD

TransformControls.htm - элемент управления,.
Геометрические элементы
ConvexBufferGeometry - буферный вариант ConvexGeometry
ConvexGeometry - создание выпуклой оболочки по заданному массиву трехмерных точек
DecalGeometry - создание сетки (mesh) с декалью
Вспомогательные элементы
FaceNormalsHelper - показывает стрелки для визуализации нормалей граней объекта
LightProbeHelper - показывает сферу для проверки (исследования) освещения на сцене
PositionalAudioHelper - показывает конус распространения звука от PositionalAudio
RectAreaLightHelper - создает видимый вспомогательный элемент для отображения зоны освещения от RectAreaLight
VertexNormalsHelper - отображает стрелки для визуализации векторов нормалей к вершинам объекта
VertexTangentsHelper - показывает стрелки для отображения векторов, касательных к вершинам объекта
Освещение
LightProbeGenerator - вспомогательный класс для создания экземпляров LightProbe
Загрузчики

BasisTextureLoader - загрузка базовых текстур в формате .basis

PRWMLoader - загрузка в формате prwm
SVGLoader - загрузка в формате svg
TGALoader - загрузка в формате tga
Объекты
