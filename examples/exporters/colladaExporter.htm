ColladaExporter

Экспортер данных в формате Collada.
Collada это формат файлов для надежного представления сцен, материалов, анимаций и другого 3D содержимого в формате xml. Этот экспортер поддерживает экспорт только геометрических элементов, материалов, текстур и иерархии сцен.

Пример

// Instantiate an exporter (создание экспортера)
var exporter = new THREE.ColladaExporter();

// Parse the input and generate the ply output
// разбор входных данных и создание выходного слоя
var data = exporter.parse( scene, null, options );
downloadFile(data);

Конструктор

ColladaExporter()
Создает новый ColladaExporter.

Методы


.parse( input, onCompleted, options )
input — экспортируемый Object3D
onCompleted — функция, которая будет вызвана по завершению экспорта. Дополнительный, необязательный параметр. The same data is immediately returned from the function.
options — параметры экспорта
  ●   version - строковое значение, которое показывает версию Collada для экспорта. Возможные значения - либо 1.4.1, либо 1.5.0. Значением по умолчанию является 1.4.1.
  ●   author - строковое значение, обозначающее имя, вносимое в поле author. По умолчанию поле author исключено.
  ●   textureDirectory - строковое значение, обозначающее путь к каталогу (относительно файла Collada), в котором хранятся текстуры.
Создает объект из файла в формате Collada и данных текстуры. Этот объект метод возвращает и передает его в функцию обратного вызова, указанную в параметре onCompleted. 
{
  // Collada file content (содержание файла в формате Collada)
  data: "",

  // List of referenced textures (список текстур)
  textures: [{
    // File directory, name, and extension of the texture data
    // каталог с файлом, имя и расширение данных текстуры
    directory: "",
    name: "",
    ext: "",

    // The texture data and original texture object
    // данные текстуры и исходный объект текстуры
    data: [],
    original: <THREE.Texture>
  }, ...]
}
