LightProbeHelper

Показывает вспомогательную сферу - датчик для проверки (исследования) освещения на сцене.

Пример

var helper = new LightProbeHelper( lightProbe, 1 );
scene.add( helper );
Другие примеры
WebGL / lightprobe / cubecamera

Конструктор

LightProbeHelper( lightProbe : LightProbe, size : Number )
lightProbe -- the light probe.
size -- размер вспомогательной сферы.

Свойства

Общие свойства смотрите в описании базового класса Mesh. 

.lightProbe : LightProbe
The light probe.

.size
Размер вспомогательной сферы.

Методы

Общие методы смотрите в описании базового класса Mesh.
