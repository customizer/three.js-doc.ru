PositionalAudioHelper

Данный вспомогательный элемент показывает конус направленности звука от объекта PositionalAudio.

Пример

var positionalAudio = new THREE.PositionalAudio( listener );
positionalAudio.setDirectionalCone( 180, 230, 0.1 );

var helper = new PositionalAudioHelper( positionalAudio );
positionalAudio.add( helper );
Другие примеры
webaudio / orientation

Конструктор

PositionalAudioHelper( audio, range )
audio - объект PositionalAudio, конус направленности звука которого нужно показать.
range - область слышимости или конус направленности звука от объекта (дополнительный, необязательный аргумент).
divisionsInnerAngle - размер внутренней части (в которой не будет уменьшения громкости) конуса направленности звука (дополнительный, необязательный аргумент).
divisionsOuterAngle - размер внешней части (в которой громкость звука будет уменьшаться) конуса направленности звука (дополнительный, необязательный аргумент).
