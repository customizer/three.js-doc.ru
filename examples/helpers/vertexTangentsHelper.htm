VertexTangentsHelper

Вспомогательный элемент показывает стрелки для визуализации векторов, касательных вершинам объекта. Требуется, чтобы касательные были указаны в атрибуте, установленном пользователем или были рассчитаны с помощью метода computeTangents.

Данный вспомогательный элемент поддерживает только объекты BufferGeometry.

Пример

var geometry = new THREE.BoxBufferGeometry( 10, 10, 10, 2, 2, 2 );
var material = new THREE.MeshNormalMaterial();
var box = new THREE.Mesh( geometry, material );

var helper = new THREE.VertexTangentsHelper( box, 1, 0x00ffff, 1 );

scene.add( box );
scene.add( helper );
Другие примеры
WebGL / helpers

Конструктор

VertexTangentsHelper( object, size, color, linewidth )
object - объект, для которого требуется показать касательные к вершинам.
size - длина стрелок (дополнительный, необязательный аргумент). Значение по умолчанию равно 1.
color - шестнадцатиричное значение цвета стрелок. Значение по умолчанию равно 0x00ffff.
linewidth - толщина линий стрелок (дополнительный, необязательный аргумент). Значение по умолчанию равно 1. (На настоящий момент настройка linewidth не поддерживается.)

Свойства

Для информации об общих свойствах посмотрите базовый класс LineSegments. 

.matrixAutoUpdate
Посмотрите описание Object3D.matrixAutoUpdate. Установите данное свойство как false, так как этот вспомогательный элемент использует matrixWorld объекта.

.object
Объект, для которого требуется показать касательные к вершинам.

.size
Длина стрелок. Значение по умолчанию равно 1.

Методы

Для информации об общих методах посмотрите базовый класс LineSegments. 

.update()
Метод обновляет предпросмотр касательных к вершинам, основываясь на изменении "мира" объекта.
