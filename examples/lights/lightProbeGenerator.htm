LightProbeGenerator

Вспомогательный класс для создания экземпляров LightProbe.

Примеры

WebGL / light probe
WebGL / light probe / cube camera

Статические методы


.fromCubeTexture( cubeTexture )
Создает пробник света (датчик для проверки освещения) из заданной (освещенности) карты окружения. Для данного метода предполагается, что карта окружения будет представлена в виде кубической текстуры.

.fromCubeRenderTarget( renderer, cubeRenderTarget )
Создает пробник света (датчик для проверки освещения) из заданной (освещенности) карты окружения. Для данного метода предполагается, что карта окружения будет представлена в виде цели кубической визуализации.
Формат цели кубической визуализации должен быть установлен как THREE.RGBAFormat.

Исходники
