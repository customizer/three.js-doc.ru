OBJLoader2Parallel

Загрузчик для загрузки ресурсов в формате obj.
Формат файла OBJ - это простой формат данных, представляющий трехмерную геометрию в удобочитаемой для человека форме, в виде положения каждой вершины, UV-координат позиции каждой текстуры, связанных с вершиной, нормалей вершин, граней, образуемых каждым многоугольником, определяемым списком вершин, и вершин текстур.

Примеры

// instantiate the loader (создание экземпляра загрузчика)
let objLoader2Parallel = new OBJLoader2Parallel();

// define where to attach the data (определение места присоединения данных)
let local = new THREE.Object3D();

// function called on successful completion of parsing
// функция, вызываемая после успешного выполнения разбора
function callbackOnLoad( object3d, message ) {
  local.add( object3d );
}

// load a resource from provided URL in parallel to Main
// загружаем ресурс из предоставленного URL параллельно основному
objLoader2Parallel.load( 'models/obj/walt/WaltHead.obj', callbackOnLoad, null, null, null );

Конструктор

OBJLoader2Parallel( manager )
manager — менеджер загрузки, используемый для загрузки. Значением по умолчанию является THREE.DefaultLoadingManager.
Создает новый OBJLoader2Parallel. Используйте его для загрузки данных в формате OBJ из файлов или для синтаксического анализа OBJ-данных из буферного массива. OBJLoader2Parallel расширяет возможности OBJLoader2 способностью запустить парсер (синтаксический анализатор) через веб-обработчик (web worker).

Общие свойства смотрите в описании базового класса OBJLoader2.

Методы

Общие методы смотрите в описании базового класса OBJLoader2. 

.parse
Смотрите описание метода OBJLoader2.parse. The callback OBJLoader2.onLoad needs to be set to be able to receive the content if used in parallel mode. Fallback is possible via OBJLoader2Parallel.setExecuteParallel.

.load
Смотрите описание метода OBJLoader2.load.

.setExecuteParallel( executeParallel )
executeParallel - значением может быть либо true, либо false.
Execution of parse in parallel via Worker is default, but synchronous OBJLoader2 parsing can be enforced via false here.

.setPreferJsmWorker ( preferJsmWorker ) : OBJLoader2Parallel
preferJsmWorker - значением может быть либо true, либо false.
Set whether jsm modules in workers should be used. This requires browser support which is currently only experimental.

.getWorkerExecutionSupport () : WorkerExecutionSupport
Allow to get hold of WorkerExecutionSupport for configuration purposes.

.buildWorkerCode () : CodeBuilderInstructions
Provide instructions on what is to be contained in the worker.
