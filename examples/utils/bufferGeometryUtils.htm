BufferGeometryUtils

Класс, содержащий вспомогательные функции для экземпляров BufferGeometry.

Методы


.computeTangents( geometry )
geometry -- экземпляр BufferGeometry, у которого должны быть атрибуты index, position, normal, и uv.
Метод вычисляет касательные и добавляет данному геометрическому элементу атрибут tangent.

.mergeBufferGeometries( geometries, useGroups )
geometries - массив экземпляров BufferGeometry.
useGroups - логическое значение, определяющее будут или нет создаваться группы для объединенного геометрического элемента.
Метод соединяет набор геометрических элементов в один экземпляр. Все геометрические элементы должны иметь совместимые атрибуты. Если объединение не получилось, метод возвращает значение null.

.mergeBufferAttributes( attributes )
attributes -- массив экземпляров BufferAttribute.
Метод соединяет набор атрибутов в один экземпляр. Все атрибуты должны иметь совместимые свойства и типы, а InterleavedBufferAttribute не поддерживается. Если объединение не получилось, метод возвращает значение null.

.interleaveAttributes( attributes )
attributes -- массив экземпляров BufferAttribute.
Чередует набор атрибутов и возвращает новый массив соответствующих атрибутов, которые сообща используют единственный экземпляр InterleavedBuffer. Все атрибуты должны быть совместимы по типу. Если объединение не получилось, метод возвращает значение null.

.estimateBytesUsed( geometry )
geometry -- экземпляр BufferGeometry для оценки используемой памяти.
Метод возвращает число байтов, используемых для представления геометрического элемента всеми атрибутами.

.mergeVertices( geometry, tolerance )
geometry -- экземпляр BufferGeometry для объединения вершин.
tolerance -- максимально допустимая разница при объединении между атрибутами вершин. Значение по умолчанию равно 1e-4 (т.е. 1×10-4 или 0,0001).
Возвращает новый объект BufferGeometry с вершинами, для которых объединены все совпадающие (в пределах допуска, т.е. определяемые параметром tolerance) атрибуты вершин.

.toTrianglesDrawMode( geometry, drawMode )
geometry -- экземпляр BufferGeometry.
drawMode -- режим рисования заданного геометрического элемента (geometry).
Возвращает новый, индексированный объект BufferGeometry на основе режима рисования THREE.TrianglesDrawMode. Этот режим соответствует WebGL примитиву gl.TRIANGLES.
