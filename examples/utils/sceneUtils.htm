SceneUtils

Класс, содержащий полезные служебные функции для работы со сценой.

Методы


.createMeshesFromInstancedMesh( instancedMesh )
instancedMesh - "экземплярная" сетка.
Метод создает новый групповой объект, который содержит новую сетку для каждого экземпляра заданной "экземплярной" сетки.

.createMultiMaterialObject( geometry, materials )
geometry - геометрический элемент, для которого создается набор материалов.
materials - материалы для объекта, указанного в geometry.
Создает новую группу, которая содержит новую сетку (mesh) для каждого материала, обозначенного в materials. Будьте осторожными и помните, что это не то же самое, что и массив материалов, устанавливающий множество материалов для одной сетки.
В основном это выгодно для объектов, которым нужен как материал, так и реализация каркаса.
