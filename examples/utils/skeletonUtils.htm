SkeletonUtils

Вспомогательные функции для управления Skeleton, SkinnedMesh и Bone.

Методы


.clone( object )
Метод клонирует данный объект и его потомков, гарантируя, что любые экземпляры SkinnedMesh будут правильно связаны с их костями. Кости также клонируются, они должны быть потомками объекта, переданного с этим методом. Остальные данные, такие как геометрические параметры объекта и материалы, используются повторно по ссылке.
