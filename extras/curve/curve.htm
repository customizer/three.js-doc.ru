Дополнения к кривым

Здесь дается описание дополнительных классов к уже имеющимся.
ArcCurve - псевдоним (алиас) для EllipseCurve.
CatmullRomCurve3 - создает гладкую трехмерную сплайновую кривую из ряда точек при помощи алгоритма Катмулл-Ром (Catmull-Rom).
CubicBezierCurve - создает гладкую двумерную кубическую кривую Безье.
CubicBezierCurve3 - создает гладкую трехмерную кубическую кривую Безье.
EllipseCurve - создает двумерную кривую в виде эллипса.
LineCurve - кривая, представляющая отрезок линии в двумерном пространстве.
LineCurve3 - кривая, представляющая отрезок линии в трехмерном пространстве.
QuadraticBezierCurve - создает гладкую двумерную квадратичную кривую Безье.
QuadraticBezierCurve3 - создает гладкую трехмерную квадратичную кривую Безье.
SplineCurve - создает гладкую сплайновую кривую из ряда точек.
