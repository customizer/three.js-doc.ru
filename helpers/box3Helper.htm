Box3Helper

вспомогательный элемент для визуализации объекта Box3 - прямоугольный параллелепипед или куб в трехмерном пространстве.

Пример

var box = new THREE.Box3();
box.setFromCenterAndSize( new THREE.Vector3( 1, 1, 1 ), new THREE.Vector3( 2, 1, 3 ) );

var helper = new THREE.Box3Helper( box, 0xffff00 );
scene.add( helper );

Конструктор

Box3Helper( box, color )
box -- Box3, который нужно показать.
color -- шестнадцатиричное значение, определяющее цвет каркаса (сторон параллелипипеда) (дополнительный, необязательный параметр). Значение по умолчанию равно 0xffff00.
Создает новый каркасный параллелепипед, представляющий переданный Box3.

Свойства

Для информации об общих свойствах посмотрите базовый класс LineSegments. 

.box
Box3, который нужно показать.

Методы

Для информации об общих методах посмотрите базовый класс LineSegments. 

.updateMatrixWorld( force )
Этот метод переопределяет метод в базовом классе Object3D, чтобы он также обновлял параллелепипед каркаса по размерам объекта, указанного в свойстве box.
