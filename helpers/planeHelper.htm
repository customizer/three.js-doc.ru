PlaneHelper

Вспомогательный элемент для визуализации объекта Plane - плоскости в трехмерном пространстве.

Пример

var plane = new THREE.Plane( new THREE.Vector3( 1, 1, 0.2 ), 3 );
var helper = new THREE.PlaneHelper( plane, 1, 0xffff00 );
scene.add( helper );

Конструктор

PlaneHelper( plane, size, color )
plane -- плоскость, которую нужно показать.
size -- длина стороны вспомогательной плоскости. Дополнительный, необязательный аргумент. Значение по умолчанию равно 1.
color -- цвет вспомогательного элемента. Дополнительный, необязательный аргумент. Значение по умолчанию равно 0xffff00.
Создает новый каркас, представляющий переданную плоскость.

Свойства

Для информации об общих свойствах посмотрите базовый класс LineSegments. 

.plane
Плоскость, которую нужно показать.

.size
Длина стороны вспомогательной плоскости.

Методы

Для информации об общих методах посмотрите базовый класс LineSegments. 

.updateMatrixWorld( force )
Этот метод переопределяет метод в базовом классе Object3D, чтобы он также обновлял вспомогательный элемент в соответствии со свойствами plane и size.
