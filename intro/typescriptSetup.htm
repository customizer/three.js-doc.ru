Установка TypeScript

three.js - это библиотека на основе JavaScript. Тем не менее, имеется возможность использовать её в проектах TypeScript, так как библиотека предоставляет файлы декларации (файлы с расширением d.ts).

Чтобы обнаружить типы three.js, компилятору TypeScript требуется минимальная конфигурация.
Понадобится установка для параметра moduleResolution значение "node", а для параметра target - значение "es6" или новее. 
// Example of minimal `tsconfig.json` file
// Пример минимального файла `tsconfig.json`
{
  "compilerOptions": {
    "target": "es6",
    "moduleResolution": "node",
  },
  "include": [ "./src/**/*.ts" ],
}
Примечание: На сегодняшний день пока невозможно использовать типизацию three.js без этих двух опций.
Примечание: Бывает, что некоторые декларации неверны и/или отсутствуют. Содействие по декларационным файлам реально приносит пользу сообществу, делая типизацию three.js лучше и точнее.
