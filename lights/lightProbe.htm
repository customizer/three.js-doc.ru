      LightProbe

Пробники света (light probes) - это альтернативный способ добавления освещения на 3D сцену. В отличие от классических источников света (например, направленное (directional), точечное (point) или прожекторное (spot) освещение), пробники света не излучают свет. Вместо этого, они сохраняют информацию о свете, проходящем через 3D-пространство. Во время визуализации свет, попадающий на 3D-объект приближается к естественному освещению реального мира с помощью данных пробника света.

Пробники света создают обычно из карт (излучения) окружающей среды. Для создания пробников света из экземпляров CubeTexture или WebGLCubeRenderTarget можно использовать класс LightProbeGenerator. Впрочем, данные об оценке освещенности могут быть представлены и в других формах, например с помощью WebXR. Это позволяет визуализировать контент дополненной реальности, который реагирует на освещение реального мира.

Текущая реализация пробника света в three.js поддерживает так называемые датчики рассеянного (диффузного) света. Этот тип светового пробника функционально эквивалентен карте излучения окружающей среды.

Примеры

WebGL / light probe
WebGL / light probe / cube camera

Конструктор


LightProbe( sh, intensity )
sh - экземпляр SphericalHarmonics3 (дополнительный, необязательный аргумент).
intensity — числовое значение силы/интенсивности пробника света (дополнительный, необязательный параметр). Значение по умолчанию равно 1.
Создает новый LightProbe.

Свойства


Смотрите базовый класс Light для общих свойств. Свойство color в настоящее время не рассчитывается и поэтому не имеет никакого влияния.
