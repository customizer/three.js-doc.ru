CompressedTextureLoader

Абстрактный базовый класс для загрузчика блочных текстур (dds, pvr, ...). Для загрузки файлов, внутри класса, используется FileLoader.

Примеры

Примеры производных классов смотрите в DDSLoader и PVRLoader.

Конструктор

CompressedTextureLoader( manager )
manager — менеджер загрузки, используемый загрузчиком. Значением по умолчанию является THREE.DefaultLoadingManager.
Создает новый CompressedTextureLoader.

Методы

Смотрите общие методы в описании базового класса Loader. 

.load( url, onLoad, onProgress, onError )
url — путь к файлу или URL-адрес файла. Также это может быть Data URI.
onLoad — функция, которая будет вызвана по завершении загрузки. Аргументом будет загруженная текстура.
onProgress — функция, которая будет вызываться пока идет процесс загрузки. Аргументом будет экземпляр XMLHttpRequest, что содержит сведения о загрузке total и loaded в байтах.
onError — функция, которая будет вызвана в случае ошибки при загрузке.
Метод начинает загрузку из источника (аргумент url) и передает загруженную текстуру в функцию, определенную в аргументе onLoad.

.setPath( path )
Метод устанавливает базовый путь или URL-адрес, с которого загружаются файлы. При загрузке множества моделей из одной и той же директории, он может быть пригодиться.
