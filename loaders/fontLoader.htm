FontLoader

Класс для загрузки шрифта в JSON-формате. Возвращает Font, являющийся массивом форм, представляющих шрифт. Для загрузки файлов, внутри класса используется FileLoader.
Конвертировать шрифты можно онлайн при помощи facetype.js.

Пример

var loader = new THREE.FontLoader();
var font = loader.load(
  // resource URL (URL-адрес ресурса)
  'fonts/helvetiker_bold.typeface.json'

  // onLoad callback (функция обратного вызова, вызываемая по окончании загрузки)
  function ( font ) {
    // do something with the font (здесь делаем что-либо с шрифтом)
    scene.add( font );
  },

  // onProgress callback (функция обратного вызова, вызываемая в процессе загрузки)
  function ( xhr ) {
    console.log( (xhr.loaded / xhr.total * 100) + '% загружено' );
  },

  // onError callback (функция обратного вызова, вызываемая при ошибке)
  function ( err ) {
    console.log( 'Произошла ошибка' );
  }
);

Конструктор

FontLoader( manager )
manager — менеджер загрузки, используемый загрузчиком. Значением по умолчанию является THREE.DefaultLoadingManager.
Создает новый FontLoader.

Методы

Смотрите общие методы в описании базового класса Loader. 

.load( url, onLoad, onProgress, onError )
url — путь к файлу или URL-адрес файла. Также это может быть Data URI.
onLoad — функция, которая будет вызвана по завершении загрузки. Аргументом будет загруженный шрифт.
onProgress — функция, которая будет вызываться пока идет процесс загрузки. Аргументом будет экземпляр XMLHttpRequest, что содержит сведения о загрузке total и loaded в байтах.
onError — функция, которая будет вызвана в случае ошибки при загрузке.
Метод начинает загрузку из источника (аргумент url) и передает загруженный шрифт в функцию, определенную в аргументе onLoad.

.parse( json )
json — разбираемая структура в формате JSON.
Метод разбирает (анализирует) структуру JSON и возвращает Font.
