Loader

Основной класс для реализации загрузчиков.

Конструктор

Loader( manager )
manager — менеджер загрузки, используемый загрузчиком. Значением по умолчанию является THREE.DefaultLoadingManager.
Создает новый Loader. Он будет вызываться как базовый класс.

Свойства


.crossOrigin
Строковое значение crossOrigin для реализации CORS для загрузки url-адреса с другого домена, который предоставляет CORS.
Значением по умолчанию является anonymous.

.manager
Менеджер загрузки, используемый загрузчиком. Значением по умолчанию является THREE.DefaultLoadingManager.

.path
Базовый путь, из которого будет загружаться основной ресурс. Значением по умолчанию является пустая строка.

.resourcePath
Базовый путь, из которого будут загружаться дополнительные ресурсы, например такие, как текстуры. Значением по умолчанию является пустая строка.

Методы


.load()
Этот метод должен реализовываться всеми конкретными загрузчиками. В нем содержится логика для загрузки основного ресурса из серверного интерфейса (бэкенда).
Примечание переводчика: Бэкенд — все, что работает на сервере, то есть «не в браузере» или «на компьютере, подсоединенном к сети (обычно к Интернету), который отвечает на сообщения от других компьютеров».
Фронтенд — все, что браузер может читать, выводить на экран и / или запускать. То есть это HTML, CSS и JavaScript.

.parse()
Этот метод должен реализовываться всеми конкретными загрузчиками. В нем содержится логика для синтаксического разбора основного ресурса на объекты three.js.

.setCrossOrigin( crossOrigin )
crossOrigin - строковое значение crossOrigin для реализации CORS для загрузки url-адреса с другого домена, который предоставляет CORS.
Устанавливает атрибут crossOrigin.

.setPath( path )
path — базовый путь, из которого будет загружаться основной ресурс. Значением по умолчанию является пустая строка.
Метод устанавливает базовый путь к основному ресурсу.

.setResourcePath( resourcePath )
resourcePath — базовый путь, из которого будут загружаться дополнительные ресурсы, например такие, как текстуры. Значением по умолчанию является пустая строка.
Метод устанавливает базовый путь к дополнительному ресурсу, такому как текстуры.
