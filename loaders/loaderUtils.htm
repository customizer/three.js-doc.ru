LoaderUtils

Объект с некоторыми вспомогательными, загрузочными функциями.

Функции


.decodeText( array )
array — поток байтов как типизированный массив.
Функция в качестве входных данных принимает поток байтов и возвращает его строковое представление.

.extractUrlBase( url )
url — url-адрес, из которого извлекается основная часть.
Извлекает базовую часть URL-адреса.
