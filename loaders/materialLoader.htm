MaterialLoader

Загрузчик для загрузки Material в формате JSON. Внутри себя для загрузки файлов он использует FileLoader.

Пример

// instantiate a loader (создаем экземпляр загрузчика)
var loader = new THREE.MaterialLoader();

// load a resource (загружаем ресурс)
loader.load(
  // resource URL (URL-адрес ресурса)
  'путь/к/material.json',

  // onLoad callback (функция обратного вызова на окончание загрузки)
  function ( material ) {
    object.material = material;
  },

  // onProgress callback (функция обратного вызова в процессе загрузки)
  function ( xhr ) {
    console.log( (xhr.loaded / xhr.total * 100) + '% загружено' );
  },

  // onError callback (функция обратного вызова на ошибки загрузки)
  function ( err ) {
    console.log( 'Произошла ошибка' );
  }
);

Конструктор

MaterialLoader( manager )
manager — менеджер загрузки, используемый загрузчиком. Значением по умолчанию является THREE.DefaultLoadingManager.
Создает новый MaterialLoader.

Свойства

Смотрите общие свойства в описании базового класса Loader. 

.textures
Объект, содержащий любые текстуры, используемые материалом. Смотрите описание метода setTextures.

Методы

Смотрите общие методы в описании базового класса Loader. 

.load( url, onLoad, onProgress, onError )
url — путь или URL-адрес к файлу. Также это может быть Data URI.
onLoad — функция, вызываемая по завершению загрузки. Её аргументом будет загруженный Material.
onProgress — функция, вызываемая во время процесса загрузки.
onError — функция, вызываемая при ошибках загрузки.
Метод начинает загрузку с указанного url-адреса (аргумент url) и возвращает объект Material, содержащий данные.

.parse( json )
json — объект в формате JSON, содержащий параметры материала.
Метод проводит синтаксический анализ (разбор) JSON-структуры и создает новый материал of the type [page:String json.type] с параметрами, определенными в объекте JSON.

.setTextures( textures )
textures — объект, содержащий любые текстуры, используемые материалом.
