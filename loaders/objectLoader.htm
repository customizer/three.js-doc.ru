ObjectLoader

Загрузчик для загрузки JSON-ресурсов в формате JSON Object/Scene.
Внутри себя для загрузки файлов он использует FileLoader.

Пример

var loader = new THREE.ObjectLoader();

loader.load(
  // resource URL (URL-адрес ресурса)
  "models/json/example.json",

  // onLoad callback (функция обратного вызова на окончание загрузки)
  // Here the loaded data is assumed to be an object 
  // здесь предполагается, что загруженные данные являются объектом
  function ( obj ) {
    // Add the loaded object to the scene (добавляем загруженный объект на сцену)
    scene.add( obj );
  },

  // onProgress callback (функция обратного вызова в процессе загрузки)
  function ( xhr ) {
    console.log( (xhr.loaded / xhr.total * 100) + '% загружено' );
  },

  // onError callback (функция обратного вызова при ошибках загрузки)
  function ( err ) {
    console.error( 'Произошла ошибка' );
  }
);


// Alternatively, to parse a previously loaded JSON structure
// Также можно проанализировать ранее загруженную JSON-структуру
var object = loader.parse( a_json_object );

scene.add( object );

Конструктор

ObjectLoader( manager )
manager — менеджер загрузки, используемый загручиком. Значением по умолчанию является THREE.DefaultLoadingManager.
Создает новый ObjectLoader.

Методы

Смотрите общие методы в описании базового класса Loader. 

.load( url, onLoad, onProgress, onError )
url — путь или URL-адрес к файлу. Также это может быть Data URI.
onLoad — функция, вызываемая по завершению загрузки. Её аргументом будет загруженный объект.
onProgress — функция, вызываемая во время процесса загрузки. Аргументом будет экземпляр XMLHttpRequest, что содержит сведения о загрузке total и loaded в байтах.
onError — функция, вызываемая при ошибках загрузки.
Метод начинает загрузку с указанного url-адреса (аргумент url) и вызывает функцию, переданную в onLoad, с разобранным содержимым ответа.

.parse( json, onLoad )
json — исходник JSON для синтаксического анализа (обязательный аргумент).
onLoad — функция, вызываемая после завершения синтаксического анализа (разбора). Её аргументом будет разобранный объект.
Метод производит синтаксический анализ содержимого json и возвращает объект threejs. Он используется внутри метода load, но также может использоваться напрямую для синтаксического анализа ранее загруженной JSON-структуры.

.parseGeometries( json )
json — исходник JSON для синтаксического анализа (обязательный аргумент).
Данный метод использует метод parse для разбора любых геометрических элементов или буферных геометрических элементов в виде JSON-структуры.

.parseMaterials( json )
json — исходник JSON для синтаксического анализа (обязательный аргумент).
Данный метод использует метод parse для разбора любых материалов в виде JSON-структуры, с помощью MaterialLoader.

.parseAnimations( json )
json — исходник JSON для синтаксического анализа (обязательный аргумент).
Данный метод использует метод parse для разбора любых анимаций в виде JSON-структуры, с помощью AnimationClip.parse.

.parseImages( json )
json — исходник JSON для синтаксического анализа (обязательный аргумент).
Данный метод использует метод parse для разбора любых объектов в виде JSON-структуры, с помощью ImageLoader.

.parseTextures( json )
json — исходник JSON для синтаксического анализа (обязательный аргумент).
Данный метод использует метод parse для разбора любых текстур в виде JSON-структуры.

.parseObject( json )
json — исходник JSON для синтаксического анализа (обязательный аргумент).
Данный метод использует метод parse для разбора любых объектов в виде JSON-структуры. Объекты могут быть следующих типов:
Scene
PerspectiveCamera
OrthographicCamera
AmbientLight
DirectionalLight
PointLight
SpotLight
HemisphereLight
Mesh
LOD
Line
LineSegments
Points
Sprite
Group
Object3D

.setCrossOrigin( value )
value — строковое значение crossOrigin для реализации CORS для загрузки url-адреса с другого домена, который предоставляет CORS.

.setTexturePath( value )
value — базовый путь или URL-адрес, с которого будут загружаться текстуры.
