LineDashedMaterial

Материал для рисования геометрических фигур в каркасном стиле пунктирными линиями.

Пример

var material = new THREE.LineDashedMaterial( {
  color: 0xffffff,
  linewidth: 1,
  scale: 1,
  dashSize: 3,
  gapSize: 1,
} );
Другие примеры
WebGL / lines / dashed

Конструктор

LineDashedMaterial( parameters )
parameters - объект с одним или несколькими свойствами, определяющими внешний вид материала (дополнительный, необязательный параметр). Здесь можно передать любое свойство материала (включая любое свойство унаследованное от LineBasicMaterial).

Свойства

Для информации об общих свойствах смотрите базовый класс LineBasicMaterial. 

.dashSize
Размер штриха, по умолчанию равен 3. Это сумма размеров самого штриха и зазора между штрихами.

.gapSize
Размер зазора (расстояния) между штрихами, по умолчанию равен 1.

.scale
Масштаб штриховой части линии. Значение по умолчанию 1.

Методы

Общие методы смотрите в описании базового класса LineBasicMaterial.
