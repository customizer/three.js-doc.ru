MeshPhysicalMaterial

Расширение MeshStandardMaterial'а, предоставляющее несколько дополнительных, физически обоснованных свойств визуализации:
Прозрачное покрытие: Некоторые материалы, вроде автомобильных красок, углепластика или мокрых поверхностей, - требуют прозрачного, отражающего слоя, нанесенного поверх другого, который может быть неровным и шероховатым. Свойство clearCoat придает материалу подобный эффект без необходимости создания отдельной прозрачной поверхности.
Физически обоснованная прозрачность: Одним из ограничений свойства opacity является уменьшение способности к отражению. Свойство transparency, основанное на физических законах, предоставляет более реалистичный выбор для тонких, прозрачных поверхностей, таких как стекло.
Расширенная способность к отражению: Более гибкая способность к отражению неметаллических материалов (свойство reflectivity).
В результате этих сложных функций затенения, MeshPhysicalMaterial имеет более низкую производительность на пиксель, чем другие материалы библиотеки three.js. Большинство эффектов отключены по умолчанию и снижают производительность вычислений по мере их включения.
Обратите внимание, что при использовании этого материала, для достижения наилучших результатов, всегда следует указывать карту окружения.

Примеры

webgl / materials / variations / physical
webgl / materials / physical / clearcoat
webgl / materials / reflectivity
webgl / materials / physical / transparency

Конструктор

MeshPhysicalMaterial( parameters )
parameters - объект с одним или несколькими свойствами, определяющими внешний вид материала (дополнительный, необязательный параметр). Здесь можно передать любое свойство материала (включая любое свойство унаследованное от Material и MeshStandardMaterial).


Свойства

Для информации об общих свойствах смотрите базовые классы Material и MeshStandardMaterial. 

.clearCoat
Уровень прозрачной оболочки, диапазон значений от 0.0 до 1.0. Значение по умолчанию равно 0.0.

.clearcoatNormalMap
Свойство может использоваться для включения собственных нормалей слоя прозрачного покрытия. Значением по умолчанию является null.

.clearcoatNormalScale
Свойство определяет насколько сильно свойство clearcoatNormalMap влияет на слой прозрачного покрытия (значение может быть в диапазоне от (0,0) до (1,1). Значение по умолчанию равно (1,1).

.clearCoatRoughness
Свойство определяет, насколько велика шероховатость прозрачной оболочки, диапазон значений от 0.0 до 1.0. Значение по умолчанию равно 0.0.

.isMeshPhysicalMaterial
Используется для проверки, является ли данный объект, или производные от него, физическим материалом сетки. По умолчанию значение равно true.
Не нужно его изменять, так как внутри three.js оно используется для оптимизации.

.defines
Объект в виде: { 'PHYSICAL': '' };. Он используется WebGLRenderer'ом для выбора шейдеров.

.reflectivity
Степень отражения, возможны значения от 0.0 до 1.0. Значение по умолчанию равно 0.5.
Свойство моделирует способность к отражению света неметаллических материалов. Оно не работает, если свойство metalness равно 1.0.

.transparency
Степень прозрачности, выбирается в диапазоне от 0.0 до 1.0. Значение по умолчанию равно 0.0.
Тонкие, прозрачные или полупрозрачные, пластиковые или стеклянные материалы в значительной степени остаются отражающими, даже если в основном они прозрачные. Свойство transparency можно использовать для моделирования таких материалов.
Когда transparency не нулевая, свойство Material.opacity должно быть установлено как 1.
