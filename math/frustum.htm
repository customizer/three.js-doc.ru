Frustum

Frustum'ы используются для определения того, что находится в поле просмотра камеры. Они помогают ускорить процесс визуализации (рендеринга) - объект, находящийся вне области просмотра камеры (в виде усеченной пирамиды), вполне можно благополучно исключить из визуализации.
Статья о усеченной пирамиде в русскоязычной Википедии.
Этот класс в основном предназначен для внутреннего использования визуализатором (рендерером) для расчета областей просмотра (в виде усеченных пирамид) камеры и теневой камеры.

Конструктор

Frustum( p0, p1, p2, p3, p4, p5 )
p0 -- дополнительный, необязательный параметр. Значением по умолчанию является новая плоскость.
p1 -- дополнительный, необязательный параметр. Значением по умолчанию является новая плоскость.
p2 -- дополнительный, необязательный параметр. Значением по умолчанию является новая плоскость.
p3 -- дополнительный, необязательный параметр. Значением по умолчанию является новая плоскость.
p4 -- дополнительный, необязательный параметр. Значением по умолчанию является новая плоскость.
p5 -- дополнительный, необязательный параметр. Значением по умолчанию является новая плоскость.
Создает новый Frustum.

Свойства


.planes
Массив из 6 плоскостей.

Методы


.clone( )
Возвращает новый Frustum с точно такими же параметрами, что и у данной.

.containsPoint( point )
point - проверяемый трехмерный вектор (Vector3).
Метод проверяет, находится ли точка (параметр point) в усеченной пирамиде области просмотра камеры (Frustum).

.copy( frustum )
frustum - копируемая усеченная пирамида (frustum).
Копирует свойства переданной frustum в данную.

.intersectsBox( box )
box - Box3 для проверки на пересечение.
Метод возвращает значение true, если параллелепипед пересекается с данной усеченной пирамидой области просмотра.

.intersectsObject( object )
Метод проверяет, не пересекается ли с данной усеченной пирамидой области просмотра (Frustum) ограничительная сфера вокруг объекта.
Обратите внимание, объект должен иметь Geometry или BufferGeometry, чтобы можно было рассчитать ограничительную сферу.

.intersectsSphere( sphere )
sphere - Sphere для проверки на пересечение.
Метод возвращает значение true, если сфера, переданная в аргументе sphere, пересекается с данной усеченной пирамидой области просмотра (Frustum).

.intersectsSprite( sprite )
Метод проверяет, не пересекается ли спрайт с данной усеченной пирамидой области просмотра (Frustum).

.set( p0, p1, p2, p3, p4, p5 )
Метод устанавливает текущую усеченную пирамиду области просмотра камеры (Frustum) из переданных плоскостей. Никакого упорядочения плоскостей не предполагается.

.setFromProjectionMatrix( matrix )
matrix - проекционная четырехмерная матрица, используемая для установки плоскостей отсечения области просмотра камеры.
Метод устанавливает усеченную пирамиду области просмотра камеры (Frustum) из проекционной матрицы.
