Plane

Двумерная поверхность, расширяемая до бесконечности в трехмерном пространстве, представленная в Hessian normal form нормальным вектором единичной длины и константой.

Конструктор

Plane( normal, constant )
normal - трехмерный вектор (Vector3) единичной длины, определяющий нормаль к плоскости. Дополнительный, необязательный аргумент. Значением по умолчанию является (1, 0, 0).
constant - знаковое расстояние (т.е. может быть как положительным, так и отрицательным) от точки начала координат до плоскости. Дополнительный, необязательный параметр, значение по умолчанию равно 0.

Методы


.applyMatrix4( matrix, optionalNormalMatrix )
matrix - применямая Matrix4.
optionalNormalMatrix - нормальная трехмерная матрица, предварительно рассчитанная из применяемой четырехмерной матрицы. Дополнительный, необязательный аргумент.
Метод применяет четырехмерную матрицу к плоскости. Матрица должна быть афинным, однородным преобразованием. Apply a Matrix4 to the plane. The matrix must be an affine, homogeneous transform. Если хочется предоставить optionalNormalMatrix, её можно создать вот так: 
var optionalNormalMatrix = new THREE.Matrix3().getNormalMatrix( matrix );

.clone( )
Возвращает новую плоскость с такими же нормалью и константой, как у этой.

.coplanarPoint( optionalTarget )
optionalTarget - если этот параметр указан, то результат будет скопирован в данный трехмерный вектор, в противном случае будет создан новый Vector3. Дополнительный, необязательный параметр.
Метод возвращает трехмерный вектор, обозначающий точку, копланарную плоскости, вычисляя проекцию вектора нормали из точки начала координат к данной плоскости.

.copy( plane )
Метод копирует значения свойств normal и constant плоскости, переданной в аргументе plane, в данную плоскость.

.distanceToPoint( point )
Возвращает расстояние (со знаком, т.е. положительное или отрицательное) от переданной точки (аргумент point) до этой плоскости.

.distanceToSphere( sphere )
Возвращает расстояние (со знаком, т.е. положительное или отрицательное) от переданной сферы (аргумент sphere) до этой плоскости.

.equals( plane )
Метод проверяет, равны ли эти (т.е. данная и приведенная в аргументе plane) две плоскости (совпадают ли их свойства normal и constant).

.intersectLine( line, optionalTarget )
line - линия, проверяемая на пересечение.
optionalTarget - если этот параметр указан, то результат будет скопирован в данный трехмерный вектор (Vector3), в противном случае будет создан новый Vector3. Дополнительный, необязательный параметр.
Метод возвращает точку пересечения линии, переданной в параметре line и данной плоскости. Если линия не пересекается с плоскостью, возвращается значение undefined - не определено. Если линия копланарна данной плоскости (т.е. линия находится на данной плоскости), возвращается начальная точка линии (start).

.intersectsBox( box )
box - параллелепипед, проверяемый на пересечение.
Определяет, пересекает или нет данная плоскость параллелепипед, переданный в аргументе box.

.intersectsLine( line )
line - линия в трехмерном пространстве, проверяемая на пересечение с плоскостью.
Метод проверяет, пересекает ли (проходит через) плоскость отрезок линии.

.intersectsSphere( sphere )
sphere - сфера, проверяемая на пересечение с плоскостью.
Определяет, пересекает или нет данная плоскость сферу, переданную в аргументе sphere.

.negate( )
Метод меняет знак на противоположный как у вектора нормали, так и у константы.

.normalize( )
Нормализует вектор нормали и соответственно подстраивает значение константы.

.projectPoint( point, optionalTarget )
point - трехмерный вектор, определяющий точку, которую нужно спроецировать на плоскость.
optionalTarget - если этот параметр указан, то результат будет скопирован в данный трехмерный вектор (Vector3), в противном случае будет создан новый Vector3. Дополнительный, необязательный параметр.
Метод проецирует точку (аргумент point) на плоскость.

.set( normal, constant )
normal - трехмерный вектор единичной длины, определяющий нормаль к плоскости.
constant - знаковое расстояние (т.е. может быть как положительным, так и отрицательным) от точки начала координат до плоскости. Значение по умолчанию равно 0.
Метод устанавливает свойства плоскости: нормаль и константу.

.setComponents( x, y, z, w )
x - x значение вектора нормали единичной длины.
y - y значение вектора нормали единичной длины.
z - z значение вектора нормали единичной длины.
w - значение свойства плоскости constant.
Метод устанавливает отдельные компоненты, определяющие плоскость.

.setFromCoplanarPoints( a, b, c )
a - первая точка на плоскости.
b - вторая точка на плоскости.
c - третья точка на плоскости.
Определяет плоскость на основе трех предоставленных точек. Предполагается, что обход точек ведется против часовой стрелки и определяется направление нормали.

.setFromNormalAndCoplanarPoint( normal, point )
normal - трехмерный вектор единичной длины, определяющий нормаль к плоскости.
point - трехмерный вектор с координатами точки.
Устанавливает свойства плоскости, определяемые по нормали и произвольной копланарной точкой.

.translate( offset )
offset - размер перемещения плоскости. the amount to move the plane by.
Перемещает плоскость на расстояние, определяемое вектором смещения (параметр offset). Обратите внимание, этот метод воздействует только на константу плоскости и не влияет на вектор нормали.
