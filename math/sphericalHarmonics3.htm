SphericalHarmonics3

Представляет сферические гармоники третьего порядка (SH). Этот класс используется пробниками света (датчиками для проверки освещения) для кодировки информации освещения.

Конструктор

SphericalHarmonics3()
Создает новый экземпляр SphericalHarmonics3.

Свойства


.coefficients
Массив, в котором хранятся 9 коэффициентов сферических гармоник (SH). Одиночный коэффициент представляет собой экземпляр трехмерного вектора.

Методы


.add( sh )
sh - сферические гармоники (SH), которые нужно добавить.
Метод добавляет заданные сферические гармоники (SH) к уже имеющемуся результату.

.addScaledSH( sh, scale )
sh - сферические гармоники (SH), которые нужно добавить.
scale - множитель масштабирования.
Удобный метод для одновременного выполнения методов add и scale.

.clone()
Метод возвращает новый экземплар SphericalHarmonics3 с равными коэффициентами.

.copy( sh )
sh - копируемые сферические гармоники (SH).
Метод копирует заданные сферические гармоники (SH) в данный экземпляр.

.equals( sh )
sh - сферические гармоники (SH), с которыми нужно сравнить уже имеющиеся.
Метод возвращает значение true, если сферические гармоники, заданные в аргументе sh и в данном экземпляре, имеют равные коэффициенты.

.fromArray( array, offset )
array - массив, содержащий числа коэффициентов сферический гармоник (SH).
offset - смещение массива (дополнительный, необязательный аргумент).
Метод устанавливает коэффициенты данного экземпляра на взятые из массива, переданного в аргументе array.

.getAt( normal, target )
normal - вектор нормали (предполагается что он единичной длины).
target - полученный (результирующий) вектор.
Метод возвращает излучение в направлении заданной нормали.

.getIrradianceAt( normal, target )
normal - вектор нормали (предполагается что он единичной длины).
target - полученный (результирующий) вектор.
Метод возвращает интенсивность излучения (radiance convolved with cosine lobe) в направлении заданной нормали.

.lerp( sh, alpha )
sh - сферические гармоники (SH) to interpolate with.
alpha - альфа-коэффициент (коэффициент интерполяции).
Выполняется линейная интерполяция между сферическими гармониками (SH), заданными в аргументе sh и данным экземпляром по заданному альфа-коэффициенту.

.scale( scale )
scale - множитель масштабирования.
Масштабирует данные сферические гармоники (SH) согласно заданного множителя масштабирования.

.set( coefficients )
coefficients - массив коэффициентов сферических гармоник (SH).
Метод устанавливает заданные коэффициенты сферических гармоник (SH) для данного экземпляра.

.toArray( array, offset )
array - целевой массив (дополнительный, необязательный аргумент).
offset - смещение массива (дополнительный, необязательный аргумент).
Метод возвращает массив с коэффициентами или копирует их в предоставленный массив. Коэффициенты представлены в виде чисел.

.zero()
Метод устанавливает все сферические гармоники (SH) равными 0.

Статические методы


.getBasisAt( normal, shBasis )
normal - вектор нормали (предполагается что он единичной длины).
shBasis - полученный базис сферических гармоник (SH).
Метод вычисляет базис сферических гармоник (SH) для заданного вектора нормали.
