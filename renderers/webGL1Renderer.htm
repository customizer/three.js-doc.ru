WebGL1Renderer

Начиная с версии r118, WebGLRenderer автоматически использует контекст визуализации (рендеринга) WebGL 2. При обновлении существующего проекта до версии r118 или более поздней, работа приложений может быть прервана по двум причинам:

  ●   Собственный код шейдера должен соответствовать GLSL 3.0.
  ●   Необходимо изменить проверки расширений WebGL 1.

Если нет времени на обновление своего кода, но все же хочется использовать последнюю версию, можно применить WebGL1Renderer. Эта версия визуализатора (рендерера) обеспечит выполнение в контексте визуализации (рендеринга) WebGL 1.

Конструктор

WebGL1Renderer( parameters )
Создает новый WebGL1Renderer.
