WebGLMultisampleRenderTarget

Специальный целевой объект визуализации, который может использоваться для применения буферов визуализации с несколькими образцами. Предостережение: WebGLMultisampleRenderTarget может использоваться только в рамках визуализации с WebGL 2.

Конструктор

WebGLMultisampleRenderTarget( width, height, options )
width - ширина целевого объекта визуализации.
height - высота целевого объекта визуализации.
options - дополнительный, необязательный аргумент. Объект, который удерживает параметры текстуры для автоматически сгенерированной целевой текстуры и логических значений depthBuffer/stencilBuffer.

Свойства


.samples
Свойство определяет число образцов, которое будет использовано для буферного хранения визуализации. Тем не менее, максимально поддерживаемый размер для мультишаблонной визуализации зависит от используемой платформы и определяется константой gl.MAX_SAMPLES.

Для данного класса доступны свойства WebGLRenderTarget

Методы


Для данного класса доступны методы WebGLRenderTarget.
