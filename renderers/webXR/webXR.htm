WebXR

WebXR (похоже это сокращение от англ. слов web eXtended Reality - расширенная реальность в сети интернет) - это группа стандартов, которые используются совместно для поддержки визуализации трехмерных сцен на оборудовании, предназначенном для представления виртуальных миров (виртуальной реальности, в англоязычной среде - virtual reality, сокращенно VR) или для добавления графических изображений в реальный мир (дополненная реальность, в англоязычной среде - augmented reality, сокращенно AR).... читать далее
API устройств WebXR реализует ядро ​​набора функций WebXR, управляя выбором выходных устройств, визуализирует трехмерную сцену на выбранном устройстве с соответствующей частотой кадров и управляет векторами движения, созданными с использованием контроллеров ввода.

Совместимые с WebXR устройства включают в себя гарнитуры полного погружения в 3D с отслеживанием движения и ориентации, очки с наложением графики поверх реальной сцены, проходящей через кадры, и портативные мобильные телефоны, которые дополняют реальность, снимая окружающий мир камерой и дополняя эту сцену изображениями, сгенерированными компьютером.

Для выполнения этих задач API WebXR-устройств предоставляет следующие ключевые возможности:
поиск совместимых устройств с выходом VR или AR
визуализация 3D сцены на устройство с соответствующей частотой кадров
отзеркаливание выхода на 2D дисплей (дополнительно, необязательно)
создание векторов, представляющих перемещение входных элементов управления.

На самом базовом уровне сцена представлена в 3D путем вычисления перспективы, применяемой к сцене, для визуализации этой сцены с точки просмотра от каждого глаза пользователя, с учетом типичного расстояния между глазами, а затем дважды визуализировать сцену, по одному разу для каждого глаза. Полученные изображения (или изображение, если сцена визуализируется дважды на одном кадре, по половине на глаз) затем отображаются пользователю.
