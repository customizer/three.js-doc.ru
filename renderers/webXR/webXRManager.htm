WebXRManager

Данный класс представляет собой обобщенный API устройства WebXR и используется внутри WebGLRenderer. WebXRManager также предоставляет общедоступный интерфейс, который позволяет пользователям включать/отключать расширенную реальность (XR) и выполнять связанные с XR задачи, такие как, например, получение контроллеров.

Свойства


.enabled
Данный флаг уведомляет визуализатор (рендерер) о готовности к визуализации расширенной реальности (XR). Значением по умолчанию является false. Установите его значение как true, если в приложении будет использоваться расширенная реальность (XR).

.isPresenting
Свойство показывает, является ли демонстрация расширенной реальности (XR) активной или нет. Значением по умолчанию является false. Этот флаг доступен только для чтения и автоматически устанавливается WebXRManager'ом.

Методы


.getController( index )
index - индекс контроллера.
Возвращает группу, представляющую так называемое пространство целевых лучей (target ray space) контроллера. Используйте это пространство для визуализации трехмерных объектов, которые помогают пользователю указывать задачи, такие как взаимодействие с пользовательским интерфейсом.

.getControllerGrip( index )
index - индекс контроллера.
Возвращает группу, представляющую так называемое пространство захвата (grip space) контроллера. Используйте это пространство, если пользователь собирается удерживать дополнительные 3D объекты, такие как световой меч.
Примечание: Если нужно показать что-то в руке пользователя и одновременно выдвинуть указывающий луч, то потребуется присоединить переносной объект к группе, возвращаемой методом getControllerGrip, а луч присоединить к группе возвращаемой методом getController. Идея в том, что возможно получить две разные группы в двух разных пространствах координат для одного и того же контроллера WebXR.

.setFramebufferScaleFactor( framebufferScaleFactor )
framebufferScaleFactor - устанавливаемый множитель масштаба буфера кадров.
Задает множитель масштабирования, используемый при определении размера буфера кадров, когда происходит визуализация на устройстве расширенной реальности (XR). Значение устанавливается относительно разрешения по умолчанию устройства расширенной реальности (XR). Значение по умолчанию для множителя равно 1. Значение 0.5 установит буфер кадров в 50% от собственного разрешения дисплея.
Примечание: При демонстрации содержимого расширенной реальности (XR) невозможно изменять множитель масштаба буфера кадров.

.setReferenceSpaceType( referenceSpaceType )
referenceSpaceType — устанавливаемый тип эталонного пространства.
Может использоваться для настройки пространственных взаимосвязей с физической средой пользователя. В зависимости от того, как пользователь перемещается в 3D-пространстве, установка соответствующего эталонного (опорного) пространства может улучшить отслеживание. Значением по умолчанию является local-floor. Ознакомьтесь с возможными значениями и случаями их использования на сайте MDN.
Примечание: Нельзя изменять тип эталонного пространства при демонстрации содержимого расширенной реальности (XR).
