DataTexture3D

Создает трехмерную текстуру. Этот тип текстуры может использоваться только при визуализации (рендеринге) с помощью WebGL 2.

Примеры

WebGL2 / materials / texture3d

Конструктор

DataTexture3D( data, width, height, depth )
data -- данные текстуры.
width -- ширина текстуры.
height -- высота текстуры.
depth -- глубина текстуры.

Свойства

Общие свойства смотрите в описании базового класса Texture. 

.wrapR
Данное свойство определяет способ укладки текстуры в направлении глубины. Значением по умолчанию является THREE.ClampToEdgeWrapping, когда край текстуры стыкуется с внешним краем текселей объекта. Также имеются ещё два варианта укладки: THREE.RepeatWrapping и THREE.MirroredRepeatWrapping. Подробности смотрите в описании констант текстуры.
Примечание переводчика: для текстур приняты следующие обозначения координат: S - горизонтальная координата (эквивалент X), T - вертикальная (эквивалент Y) и R - в глубину (Z).
